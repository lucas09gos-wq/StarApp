# StarApp

Aplicativo mobile desenvolvido para ajudar estudantes a organizar tarefas, provas e horários de estudo de forma simples e eficiente.

## Objetivo

O aplicativo busca solucionar a dificuldade que muitos estudantes possuem em organizar seus estudos. Com ele, é possível planejar atividades, acompanhar prazos, receber lembretes e manter uma rotina de estudos mais produtiva.

## Funcionalidades

* Agenda de tarefas e provas
* Cronograma de estudos personalizado
* Controle de matérias e atividades
* Notificações e lembretes automáticos
* Cadastro e gerenciamento de planejamento de estudos
* Upload de fotos de atividades
* Login e gerenciamento de conta

## Tecnologias utilizadas

* React Native
* Expo
* Expo Router
* JavaScript
* AsyncStorage
* Expo Notifications

## Como instalar

```bash
npm install
```

## Como executar

```bash
npx expo start
```

## Como gerar o APK

```bash
npx eas build:configure
npx eas build -p android --profile preview
```
